## :fire: How to use


1. Make a clone :

```sh
  $ git clone https://github.com/pablomagalhaes/crm-addi.git
```

2. Running the Application:

```sh
  # Install the dependencies
  $ cd crm-addi
  $ yarn install --ignore-engines

  ## Running / Development
  $ yarn start:local

  ## Building
  $ yarn build:local
  $ yarn build:production

  # Testing
  $ yarn test:unit
 
```
